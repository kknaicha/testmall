```
npm install
```

```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Customize configuration
See [Configuration Reference](https://cli.vuejs.org/config/).

1-  划分目录结构 建文件夹。

2-  引入normalize.css 对标签进行统一

3- 创建vue.config.js 起别名，粘贴.editorconfig 控制统一的代码风格。
